นภัสกร แซ่เนี้ยว ม.5 

สวัสดีครับพี่ ผมจะสอนใช้โปรแกรมของผมครับ

กดดับเบื้ลคลิ๊ก ตัวไฟล์ Data.py ได้เลยครับ ในตัวCode ผมไม่ได้ Comment 
เพราะถ้าเขียบนอธิบายหมดมันจะยาวมากๆเลยนะครับ ผมเลย Comment ส่วนที่เป็น
Highlight Process เท่ๆไว้ครับ

ในตัว โปรแกรม จะมี 2 Function ให้เลือก
1. เครื่องคิดเลขอย่างง่าย 1 Oparetor pertime
เช่น 1+1 2+2 2**3 ใช้ตัวดำเนินการเพียง 1 ตัว

Input เป็น 1 + 1 : <ตัวเลข> <ตัวดำเนินการ> <ตัวเลข> ขั้นด้วยช่องว่าง
Output จะได้เป็นผลลัพธ์จากการคำนวณ

2. ตัวแปลงเป็นอุณหภูมิระหว่างหน่วย

Input เป็น 25C = F : <ตัวเลข><หน่วยอุณหภูมิ> = <หน่วยอุณหภูมิที่ต้องการแปลง>
Output จะได้เป็นค่าของอุณหภูมิหน่อยใหม่

ส่วนตัว log.txt เป็นตัวไฟล์แสดงประวัติการ Input และ Output 
ของตัว Program หรือ Data.py ครับ

ขอบคุณครับบบบบ ขอใบ Certificate ด้วยนะครับพี่่่่ ผมอยากเข้าลาดกระบังจริงๆ
